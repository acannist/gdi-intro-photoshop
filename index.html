Why Photoshop?

Raster vs. Vector

Raster:

  • Dot / pixel based
  • Cannot resize without quality loss
  • Precise editing, modify each pixel
  • GIF, JPG, PNG
  • Photoshop

Vector:

  • Mathematical calculations that form objects or lines
  • No pixels therefore resolution-independent
  • Infinitely resize while maintaining quality
  • SVG, EPS
  • Illustrator, Sketch

Canvas Size vs. Image Size

  • Canvas Size
    • Changes the current size of your document, but not the image itself
    • The borders expand or contract without the image
    • Adding to the canvas size will add blank area around the image
    • Decreasing the canvas size will crop the image, meaning you will lose some of the image depending on where you crop it
  • Image Size
    • When you modify the image size, you change how large the image is
    • Use image size when you want to shrink, expand, or change the resolution of an image
    • The image borders “stick” to the image and the image is pulled or squished with the borders

Optimization

Saving Your Files in Photoshop

  • Save for Web
    • You can use the built-in Save for Web feature by going to File > Save for Web
    • Preview Images in the Dialog Box using multiple tabs
    • View the optimized image information and download time
    • Fine tune using the settings
  • Export As
    • Newer versions of Photoshop use File > Export > Export As
    • This has more options to choose from and more dialogs to help you optimize
    • Export As even allows for SVG
